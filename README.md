# ColoxAdminMobile / 云端后台

当前版本：1.2.0-ui-clean

改动：
- 首页改为清爽数据概览：用户总数、今日新增、本周新增、专业版、永久版、在线设备。
- 移除重复快捷入口，避免首页拥挤。
- 最近用户保持一行列表，点击进入详情。
- 用户管理页为一行一条列表，只显示核心信息，完整信息进入详情查看。
- 桌面显示名：云端后台；APP 内标题：云端用户管理器 Pro。


## v1.2.2

- 修复“记住登录状态”：勾选后再次打开 APP 会直接进入首页。
- 点击“我的 > 锁定后台”会清除记住登录状态。


## v1.2.3

- 登录页改为“记住密码”：下次打开仍显示登录页，但密码会自动填入，点击登录即可。
- 不再自动跳过登录进入首页。

## 本次修复
- APP 所有时间显示统一按北京时间处理。
- 手动修改到期时间时，输入框填写北京时间，例如 `2026-06-30 23:59`。
- 保存到云端时会自动转换为云端需要的 UTC ISO 值，重新读取后仍显示为北京时间，避免 11:15 变 19:15。
